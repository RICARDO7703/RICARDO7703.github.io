Bucles

Seleccione todas las definiciones correctas:

Un bucle es una estructura de control en programación que permite repetir un bloque de código varias veces. - Correcta
Un bucle es una estructura de control en programación que permite realizar operaciones matemáticas complejas.
El bucle while es una estructura de bucle en la que se repite un bloque de código mientras se cumpla una determinada condición. - Correcta
El bucle for es una estructura de bucle en la que se repite un bloque de código un número fijo de veces, independientemente de cualquier condición.

Vectores

¿Qué es un vector?

Un vector es una representación gráfica de una dirección en el espacio tridimensional.
Un vector es una operación matemática que se utiliza para multiplicar matrices.
Un vector es una estructura de datos que almacena una colección de elementos del mismo tipo. - Correcta
